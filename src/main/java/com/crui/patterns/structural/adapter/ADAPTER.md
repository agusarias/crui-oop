Osea que actua como un traductor que permite a la clase existente que no puede
modificar su clase y el codigo que necesitas usar.

conectar en el unico toma que hay en el lugar. Este toma recibe enchufes de tres
patas. Para solucionar esto, se usa un adaptador que ermite recibir un enchufe de dos patatas con una salida de tres patas.

Lo que voy hacer, es generar una clase que se utilice como tipo de dato.
